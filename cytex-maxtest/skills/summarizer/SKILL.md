# Summarizer

Use this skill when the user asks to summarize a document or article.

## Steps

1. Read the provided text.
2. Produce 3-5 concise bullet points capturing the key ideas.
3. Keep the summary under 100 words unless the user asks for more.

## Rules

- Do not invent facts that are not in the source text.
- Preserve numbers and named entities exactly.
